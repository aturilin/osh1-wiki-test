# Проект: мобильное приложение

Разработка мобильного приложения ОШ-1 для iOS и Android.

## Цель

Дать ученикам доступ к урокам, расписанию, ДЗ и чату с куратором через нативное приложение — без необходимости заходить в браузер.

## Фазы

### Phase 1 — прототип (Q1 2026) — **в работе**
- Базовая навигация
- Просмотр расписания
- Push-уведомления
- Чат с куратором (интеграция с Telegram)

### Phase 2 — MVP (Q2 2026)
- Просмотр уроков (стриминг + кэш)
- Сдача ДЗ через приложение
- Оплата из приложения

### Phase 3 — полный запуск (Q3-Q4 2026)
- Видео-уроки в высоком качестве
- Геймификация
- AI-помощник по ДЗ
